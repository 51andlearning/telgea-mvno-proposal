# Telgea MVNO Proposal Site

A standalone proposal website for the DSG Digital Mobile Platform licensing offer to Telgea.

Built with Next.js 15 (App Router), TypeScript, Tailwind CSS, and shadcn UI.

## Quick Start

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
pnpm build
pnpm start
```

See [docs/DEPLOYMENT.md](docs/DEPLOYMENT.md) for full Vercel deployment instructions.
