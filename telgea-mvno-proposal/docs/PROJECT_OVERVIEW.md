# Project Overview

## What This Is

A standalone, publicly accessible proposal website built for **Digital Solutions Group (DSG)** to present the MVNO licensing offer for the DSG Digital Mobile Platform to **Telgea**.

The site replaces the static Word/PDF proposal with an interactive, branded web experience — structured to guide Telgea stakeholders through the proposal in a logical flow: context → scope → commercial terms → T&Cs → acceptance.

## Target Audience

- Telgea decision-makers and commercial leads
- Internal DSG stakeholders reviewing the proposal before sending

## Deployment Environment

- **Hosting:** Vercel (hobby or pro tier)
- **Framework:** Next.js 15 (App Router)
- **Domain:** To be configured post-deployment (custom domain via Vercel DNS or CNAME)
- **Environment:** Production only at launch; no staging environment initially
