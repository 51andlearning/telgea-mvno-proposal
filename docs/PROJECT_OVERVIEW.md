# Project Overview

## What This Site Is

A standalone, publicly deployable proposal website presenting the DSG Digital Mobile Platform MVNO offering to Telgea. It converts the confidential PDF proposal (DSG Digital Mobile MVNO Proposal Telgea 20260327 v2) into an interactive, web-native experience.

The site is intended to:
- Present the proposal narrative in a readable, section-by-section format
- Surface key commercial terms, financials, and timelines clearly
- Allow Telgea stakeholders to navigate the proposal without opening a PDF
- Serve as a credibility asset for DM / DSG during the proposal acceptance period

## Target Audience

Primary: Telgea's authorised advisors and decision-makers evaluating the MVNO partnership.
Secondary: Internal DM / DSG stakeholders reviewing the live proposal asset.

This is a **confidential document**. All pages should carry appropriate confidentiality labelling. The site should not be indexed by search engines (use `noindex` meta tags in production).
